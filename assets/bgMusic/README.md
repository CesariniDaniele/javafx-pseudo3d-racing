# 🎵 Background Music

> ⚠️ **Note:** Audio files are not included in this repository for copyright reasons.

However, you can add your own audio files to enable music in the game.

Expected filenames:

- `RecklessRoboGangJohnny.mp3` → welcome screen
- `BikeChase.mp3` → race music
- `victory.mp3` → victory screen
- `gameOver.mp3` → game over screen

Place them inside: `assets\bgMusic\`

The game will run correctly even without these files.
